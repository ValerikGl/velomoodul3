# Velo – Smart Urban Mobility

Front-end project created as part of **III Moodul**.

Velo is a modern electric mobility platform that allows users to discover, compare and book electric scooters, e-bikes and other urban transport solutions through a responsive web application.

## Installation

Clone the repository:

```bash
git clone https://github.com/ValerikGl/velomoodul3.git
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
